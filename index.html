Game Development Journey

Game Development Journey

2018

Started my journey with tynker indie game projects and foundational learning in game design. Around this time I would start to practice Photoshop and Premiere Pro and learning pixel art. I also created a Fiverr store page for editing game advertisements and made some of my first money.

2020

Around this time during quarantine I opened Unreal Engine for the first time, starting smaller practice projects using blueprints learning the engine in-and out.

2022

Finally I decided to get serious about finishing a project and making something that I've always dreamed of playing, this game would become SlimeClimb. This was entire experience all about refining my expertise, and executing my vision. Around this time is when I first heard of LLM models and became fascinated with artifical intelegence and started researching the topic.

2024-Now

After much planning and work, I was finally able to release a relatively successful open-beta test to the public, gaining around 500 players between both platforms. In this time I also decided to start the beginings of creating my own sort of "engine inside of Unreal Engine" to make my game's level builder system to encourage community made content.

Now I'm in the process today finishing my game for the first time and actually releasing the project. Since I started I've learned so much about iOS SDK, Android SDK, UI/UX, packaging for Appstore Connect/GooglePlay Console, pc game development, and especially taking and implimenting feedback from beta testers.
